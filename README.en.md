## Credits & takedown (be decent)
- We prefer original links + author credits.
- Unattributed items are allowed but will be clearly marked as `Author: Unknown` + `needs-credit`.
- **Preview images / short GIFs / thumbnails (optional)** (see `docs/8_media_policy.md`):
  - Only include previews when the submitter confirms they are the author / have permission / embedding is allowed by the platform.
  - We always keep the original source link as the primary reference.
- Takedown: submit a **Removal Request** issue, we will respond fast.
