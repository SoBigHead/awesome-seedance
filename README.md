## 版权与署名（厚道规则）
- 我们优先收录「原链接 + 原作者署名」。
- 允许先收录来源不明的案例/提示词，但会显著标注 `Author: Unknown` + `needs-credit`，并欢迎社区补充出处。
- 如原作者希望下架：请提交 **Removal Request**（Issue），我们会快速处理。

## 如何投稿
- 提交案例 / prompt / 模板：请用 GitHub Issues 表单。
- 补充署名（给出原作者链接）：请用 “Credit Fix” 表单。
